    A Programming Language

    Two Variables

    • x, y

    Three Operations

    • x++
    • x--
    • x = 0 ? L1 : L2







    Example Program

    (What does it do?)

    L1: x++
    y--
    y = 0 ? L2 : L1
L2: ...







    The above language is “equivalent to” every PL!

    (See wikipedia)

    • Fortnite
    • Spotify

    QuickSort in C

    void sort(int arr[], int beg, int end){
  if (end > beg + 1){
    int piv = arr[beg];
    int l = beg + 1;
    int r = end;
    while (l != r-1)
       if(arr[l] <= piv) l++;
       else swap(&arr[l], &arr[r--]);
    if(arr[l]<=piv && arr[r]<=piv)
       l=r+1;
    else if(arr[l]<=piv && arr[r]>piv)
       {l++; r--;}
    else if (arr[l]>piv && arr[r]<=piv)
       swap(&arr[l++], &arr[r--]);
    else r=l-1;
    swap(&arr[r--], &arr[beg]);
    sort(arr, beg, r);
    sort(arr, l, end);
  }
}







    QuickSort in Haskell

    sort []     = []
sort (x:xs) = sort ls ++ [x] ++ sort rs
  where
    ls      = [ l | l <- xs, l <= x ]
    rs      = [ r | r <- xs, x <  r ]
